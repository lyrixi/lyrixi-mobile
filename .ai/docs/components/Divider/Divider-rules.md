# Divider Rules

> 源文档：`src/components/Divider/index.zh-CN.md`。更新后请执行 `npm run build:ai-docs`。

## 必须使用
- 从 `lyrixi-mobile` 引入 `Divider`，**禁止**自造同类 UI。
- 子组件写法：`Divider.Sub`（与库导出一致）。

## 何时使用
- 需要分隔内容时
- 需要显示分割线时
- 需要在内容之间添加分隔时
