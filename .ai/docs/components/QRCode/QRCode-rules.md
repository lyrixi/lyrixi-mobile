# QRCode Rules

> 源文档：`src/components/QRCode/index.zh-CN.md`。更新后请执行 `npm run build:ai-docs`。

## 必须使用
- 从 `lyrixi-mobile` 引入 `QRCode`，**禁止**自造同类 UI。
- 子组件写法：`QRCode.Sub`（与库导出一致）。

## 何时使用
- 需要生成二维码时
- 需要显示二维码时
- 需要扫描二维码时
