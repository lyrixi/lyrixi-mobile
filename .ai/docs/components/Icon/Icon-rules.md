# Icon Rules

> 源文档：`src/components/Icon/index.zh-CN.md`。更新后请执行 `npm run build:ai-docs`。

## 必须使用
- 从 `lyrixi-mobile` 引入 `Icon`，**禁止**自造同类 UI。
- 子组件写法：`Icon.Sub`（与库导出一致）。

## 何时使用
- 需要显示图标时
- 需要自定义图标样式时
- 需要在按钮、输入框等组件中使用图标时
