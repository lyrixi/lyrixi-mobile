# List Rules

> 源文档：`src/components/List/index.zh-CN.md`。更新后请执行 `npm run build:ai-docs`。

## 必须使用
- 从 `lyrixi-mobile` 引入 `List`，**禁止**自造同类 UI。
- 子组件写法：`List.Sub`（与库导出一致）。

## 何时使用
- 需要展示数据列表时
- 需要支持单选或多选时
- 需要分组显示数据时
