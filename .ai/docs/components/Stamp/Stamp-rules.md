# Stamp Rules

> 源文档：`src/components/Stamp/index.zh-CN.md`。更新后请执行 `npm run build:ai-docs`。

## 必须使用
- 从 `lyrixi-mobile` 引入 `Stamp`，**禁止**自造同类 UI。
- 子组件写法：`Stamp.Sub`（与库导出一致）。

## 何时使用
- 需要显示加载占位效果时
- 需要在内容加载时显示骨架屏时
- 需要提升用户体验时
