# Tooltip Rules

> 源文档：`src/components/Tooltip/index.zh-CN.md`。更新后请执行 `npm run build:ai-docs`。

## 必须使用
- 从 `lyrixi-mobile` 引入 `Tooltip`，**禁止**自造同类 UI。
- 子组件写法：`Tooltip.Sub`（与库导出一致）。

## 何时使用
- 需要显示提示信息时
- 需要在元素上显示提示时
- 需要显示帮助信息时
