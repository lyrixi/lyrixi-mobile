# Steps Rules

> 源文档：`src/components/Steps/index.zh-CN.md`。更新后请执行 `npm run build:ai-docs`。

## 必须使用
- 从 `lyrixi-mobile` 引入 `Steps`，**禁止**自造同类 UI。
- 子组件写法：`Steps.Sub`（与库导出一致）。

## 何时使用
- 需要显示步骤流程时
- 需要显示进度步骤时
- 需要引导用户完成操作时
