# Button Rules

> 源文档：`src/components/Button/index.zh-CN.md`。更新后请执行 `npm run build:ai-docs`。

## 必须使用
- 从 `lyrixi-mobile` 引入 `Button`，**禁止**自造同类 UI。
- 子组件写法：`Button.Sub`（与库导出一致）。

## 何时使用
- 需要触发一个操作时
- 需要提交表单时
- 需要执行确认、取消等操作时

## 子组件
- `Button.Text`
