# AttachUploader Rules

> 源文档：`src/components/AttachUploader/index.zh-CN.md`。更新后请执行 `npm run build:ai-docs`。

## 必须使用
- 从 `lyrixi-mobile` 引入 `AttachUploader`，**禁止**自造同类 UI。
- 子组件写法：`AttachUploader.Sub`（与库导出一致）。

## 何时使用
- 需要上传附件文件时
- 需要管理附件上传状态时
- 需要显示附件上传进度时
