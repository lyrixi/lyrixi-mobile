# Picker Rules

> 源文档：`src/components/Picker/index.zh-CN.md`。更新后请执行 `npm run build:ai-docs`。

## 必须使用
- 从 `lyrixi-mobile` 引入 `Picker`，**禁止**自造同类 UI。
- 子组件写法：`Picker.Sub`（与库导出一致）。

## 何时使用
- 需要从多个选项中选择时
- 需要滚动选择时
- 需要选择日期、时间等时
