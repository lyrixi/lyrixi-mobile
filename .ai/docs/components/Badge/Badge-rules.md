# Badge Rules

> 源文档：`src/components/Badge/index.zh-CN.md`。更新后请执行 `npm run build:ai-docs`。

## 必须使用
- 从 `lyrixi-mobile` 引入 `Badge`，**禁止**自造同类 UI。
- 子组件写法：`Badge.Sub`（与库导出一致）。

## 何时使用
- 需要显示数字标记时
- 需要显示状态标记时
- 需要在元素上显示标记时
