# Flex Rules

> 源文档：`src/components/Flex/index.zh-CN.md`。更新后请执行 `npm run build:ai-docs`。

## 必须使用
- 从 `lyrixi-mobile` 引入 `Flex`，**禁止**自造同类 UI。
- 子组件写法：`Flex.Sub`（与库导出一致）。

## 何时使用
- 需要创建灵活的布局时
- 需要控制元素对齐方式时
- 需要控制元素间距时

## 子组件
- `Flex.Compact`
