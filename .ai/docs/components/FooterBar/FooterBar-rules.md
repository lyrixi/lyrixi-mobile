# FooterBar Rules

> 源文档：`src/components/FooterBar/index.zh-CN.md`。更新后请执行 `npm run build:ai-docs`。

## 必须使用
- 从 `lyrixi-mobile` 引入 `FooterBar`，**禁止**自造同类 UI。
- 子组件写法：`FooterBar.Sub`（与库导出一致）。

## 何时使用
- 需要显示页面底部内容时
- 需要显示操作按钮时
- 需要在页面底部显示固定内容时

## 子组件
- `FooterBar.Button`
