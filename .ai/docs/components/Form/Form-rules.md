# Form Rules

> 源文档：`src/components/Form/index.zh-CN.md`。更新后请执行 `npm run build:ai-docs`。

## 必须使用
- 从 `lyrixi-mobile` 引入 `Form`，**禁止**自造同类 UI。
- 子组件写法：`Form.Sub`（与库导出一致）。

## 何时使用
- 需要收集用户输入数据时
- 需要对输入进行校验时
- 需要提交表单数据时

## 子组件
- `Form.Item`
- `Form.useForm`
- `Form.useWatch`
