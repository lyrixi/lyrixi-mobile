# TabBar Rules

> 源文档：`src/components/TabBar/index.zh-CN.md`。更新后请执行 `npm run build:ai-docs`。

## 必须使用
- 从 `lyrixi-mobile` 引入 `TabBar`，**禁止**自造同类 UI。
- 子组件写法：`TabBar.Sub`（与库导出一致）。

## 何时使用
- 见 `src/components/TabBar/index.zh-CN.md` 中「何时使用」。
