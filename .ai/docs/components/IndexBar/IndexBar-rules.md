# IndexBar Rules

> 源文档：`src/components/IndexBar/index.zh-CN.md`。更新后请执行 `npm run build:ai-docs`。

## 必须使用
- 从 `lyrixi-mobile` 引入 `IndexBar`，**禁止**自造同类 UI。
- 子组件写法：`IndexBar.Sub`（与库导出一致）。

## 何时使用
- 需要快速定位内容时
- 需要显示索引导航时
- 需要字母索引时

## 子组件
- `IndexBar.Anchor`
