# Combo Rules

> 源文档：`src/components/Combo/index.zh-CN.md`。更新后请执行 `npm run build:ai-docs`。

## 必须使用
- 从 `lyrixi-mobile` 引入 `Combo`，**禁止**自造同类 UI。
- 子组件写法：`Combo.Sub`（与库导出一致）。

## 何时使用
- 需要自定义渲染组合元素时
- 需要创建可点击的组合元素时
