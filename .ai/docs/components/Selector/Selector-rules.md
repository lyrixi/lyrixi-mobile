# Selector Rules

> 源文档：`src/components/Selector/index.zh-CN.md`。更新后请执行 `npm run build:ai-docs`。

## 必须使用
- 从 `lyrixi-mobile` 引入 `Selector`，**禁止**自造同类 UI。
- 子组件写法：`Selector.Sub`（与库导出一致）。

## 何时使用
- 需要从多个选项中选择时
- 需要单选或多选时
- 需要网格布局显示选项时
