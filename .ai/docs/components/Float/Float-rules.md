# Float Rules

> 源文档：`src/components/Float/index.zh-CN.md`。更新后请执行 `npm run build:ai-docs`。

## 必须使用
- 从 `lyrixi-mobile` 引入 `Float`，**禁止**自造同类 UI。
- 子组件写法：`Float.Sub`（与库导出一致）。

## 何时使用
- 需要显示悬浮按钮时
- 需要可拖动的按钮时
- 需要固定在页面上的按钮时
