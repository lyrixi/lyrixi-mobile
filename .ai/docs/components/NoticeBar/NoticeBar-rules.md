# NoticeBar Rules

> 源文档：`src/components/NoticeBar/index.zh-CN.md`。更新后请执行 `npm run build:ai-docs`。

## 必须使用
- 从 `lyrixi-mobile` 引入 `NoticeBar`，**禁止**自造同类 UI。
- 子组件写法：`NoticeBar.Sub`（与库导出一致）。

## 何时使用
- 需要显示通知、提示、警告等信息时
- 需要在页面顶部或内容区域显示重要信息时
- 需要吸引用户注意但不强制交互时
