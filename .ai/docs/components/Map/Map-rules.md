# Map Rules

> 源文档：`src/components/Map/index.zh-CN.md`。更新后请执行 `npm run build:ai-docs`。

## 必须使用
- 从 `lyrixi-mobile` 引入 `Map`，**禁止**自造同类 UI。
- 子组件写法：`Map.Sub`（与库导出一致）。

## 何时使用
