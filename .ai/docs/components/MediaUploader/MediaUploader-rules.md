# MediaUploader Rules

> 源文档：`src/components/MediaUploader/index.zh-CN.md`。更新后请执行 `npm run build:ai-docs`。

## 必须使用
- 从 `lyrixi-mobile` 引入 `MediaUploader`，**禁止**自造同类 UI。
- 子组件写法：`MediaUploader.Sub`（与库导出一致）。

## 何时使用
- 需要上传图片、视频时
- 需要管理媒体上传状态时
- 需要显示媒体上传进度时
