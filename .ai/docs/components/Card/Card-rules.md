# Card Rules

> 源文档：`src/components/Card/index.zh-CN.md`。更新后请执行 `npm run build:ai-docs`。

## 必须使用
- 从 `lyrixi-mobile` 引入 `Card`，**禁止**自造同类 UI。
- 子组件写法：`Card.Sub`（与库导出一致）。

## 何时使用
- 需要展示相关内容区块时
- 需要将信息分组展示时
- 需要带标题和内容的结构化展示时

## 子组件
- `Card.Header`
- `Card.Main`
