# Switch Rules

> 源文档：`src/components/Switch/index.zh-CN.md`。更新后请执行 `npm run build:ai-docs`。

## 必须使用
- 从 `lyrixi-mobile` 引入 `Switch`，**禁止**自造同类 UI。
- 子组件写法：`Switch.Sub`（与库导出一致）。

## 何时使用
- 需要表示开关状态时
- 需要切换两种状态时
- 需要表示启用/禁用状态时
