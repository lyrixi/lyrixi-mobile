# Result Rules

> 源文档：`src/components/Result/index.zh-CN.md`。更新后请执行 `npm run build:ai-docs`。

## 必须使用
- 从 `lyrixi-mobile` 引入 `Result`，**禁止**自造同类 UI。
- 子组件写法：`Result.Sub`（与库导出一致）。

## 何时使用
- 需要显示操作结果时
- 需要显示空状态时
- 需要显示错误状态时
