# Modal Rules

> 源文档：`src/components/Modal/index.zh-CN.md`。更新后请执行 `npm run build:ai-docs`。

## 必须使用
- 从 `lyrixi-mobile` 引入 `Modal`，**禁止**自造同类 UI。
- 子组件写法：`Modal.Sub`（与库导出一致）。

## 何时使用
- 需要显示对话框时
- 需要用户确认操作时
- 需要显示重要信息时
