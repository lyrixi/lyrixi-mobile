# Transfer Rules

> 源文档：`src/components/Transfer/index.zh-CN.md`。更新后请执行 `npm run build:ai-docs`。

## 必须使用
- 从 `lyrixi-mobile` 引入 `Transfer`，**禁止**自造同类 UI。
- 子组件写法：`Transfer.Sub`（与库导出一致）。

## 何时使用
- 需要在两栏之间移动元素时
- 需要选择多个选项时
- 需要管理已选和未选项时
