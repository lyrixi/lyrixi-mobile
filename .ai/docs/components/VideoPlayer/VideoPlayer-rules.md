# VideoPlayer Rules

> 源文档：`src/components/VideoPlayer/index.zh-CN.md`。更新后请执行 `npm run build:ai-docs`。

## 必须使用
- 从 `lyrixi-mobile` 引入 `VideoPlayer`，**禁止**自造同类 UI。
- 子组件写法：`VideoPlayer.Sub`（与库导出一致）。

## 何时使用
- 需要播放视频时
- 需要显示视频播放器时
- 需要控制视频播放时
