# Checkbox Rules

> 源文档：`src/components/Checkbox/index.zh-CN.md`。更新后请执行 `npm run build:ai-docs`。

## 必须使用
- 从 `lyrixi-mobile` 引入 `Checkbox`，**禁止**自造同类 UI。
- 子组件写法：`Checkbox.Sub`（与库导出一致）。

## 何时使用
- 需要从一组选项中选择多个时
- 需要表示开关状态时
- 需要确认操作时

## 子组件
- `Checkbox.Group`
