# Accordion Rules

> 源文档：`src/components/Accordion/index.zh-CN.md`。更新后请执行 `npm run build:ai-docs`。

## 必须使用
- 从 `lyrixi-mobile` 引入 `Accordion`，**禁止**自造同类 UI。
- 子组件写法：`Accordion.Sub`（与库导出一致）。

## 何时使用
- 需要折叠和展开内容时
- 需要显示可折叠的面板时
- 需要节省页面空间时
